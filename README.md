## üß† Technical Architecture

FinSight AI is built on a modern, high-performance stack optimized for intelligence and aesthetics:

- **Frontend**: React 19 + TypeScript + Tailwind CSS
- **Intelligence Engine**: Google Gemini API
    - **Gemini 3 Pro**: For complex reasoning and deep financial advice.
    - **Gemini 3 Flash**: For low-latency simulation logic and chat interactions.
    - **Gemini 2.5 Flash TTS**: Powering Finny's voice interaction.
- **Grounding**: Utilizing the `googleSearch` tool for real-time market verified data.
- **Visualization**: Recharts for institutional-grade financial modeling.

## üíª Getting Started

### Local Development
1. Clone the repository:
   ```bash
   git clone https://github.com/mrbrightsides/finsight.git
   ```
2. Set up your environment variable:
   Create a `.env` file and add your Google Gemini API Key:
   ```env
   API_KEY=your_gemini_api_key_here
   ```
3. Install dependencies and start the dev server:
   ```bash
   npm install
   npm run dev
   ```
